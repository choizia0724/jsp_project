jsp_project는 springboot와 kendo UI로 만든 게시판 웹프로그램입니다.

Jwt를 쿠키에 저장하여 인증방식을 사용하였습니다.

회원가입시 아이디중복체크

로그인시 아이디 비밀번호 유효체크 

등의 기능이 포함되어있습니다.

MongoDB를 사용하였으며 JPA를 사용해 가독성있는 코드로 구성되어있습니다.
