<b>JSP PROJECT는 SpringBoot와 Kendo UI로 만든 게시판 웹프로그램입니다.</b>

<ol>
<li>Jwt를 쿠키에 저장하여 인증방식</li>

<li>회원가입시 아이디중복체크</li>

<li>로그인시 아이디 비밀번호 유효체크 </li>
</ol>
등의 기능이 포함되어있습니다.
